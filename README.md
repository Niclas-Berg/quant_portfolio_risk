# quant_portfolio_risk

Professional Monte Carlo Portfolio Risk Simulator

Overview
- A production-style Python project that fetches historical prices, estimates parameters, and runs Monte Carlo simulations using Geometric Brownian Motion (GBM) with Cholesky decomposition to model correlations.
- Computes 95% and 99% Value at Risk (VaR) and Conditional VaR (CVaR).
- Saves a high-quality visualization in `src/portfolio_risk_chart.png`.

Quantitative Models
- Geometric Brownian Motion (GBM): GBM models asset price dynamics as
  $dS_t = \mu S_t dt + \sigma S_t dW_t$
  where $\mu$ is the drift, $\sigma$ is volatility, and $dW_t$ is a Wiener increment. Over a horizon $T$, the closed-form solution for prices is
  $S_T = S_0 \exp\left((\mu - 0.5\sigma^2)T + \sigma \sqrt{T}Z\right)$
  with $Z$ a standard normal random variable.

- Cholesky Decomposition for Correlation: To simulate correlated asset returns we use the asset return covariance matrix which is decomposed via Cholesky: given a correlation matrix $\rho$ and vector of marginal volatilities, correlated normals are generated by
  $Z_{corr} = L Z$, where $L$ is the lower-triangular Cholesky factor of $\rho$.

Usage
1. Create and activate the virtual environment (instructions below).
2. Run the simulator: `./.venv/bin/python src/portfolio_simulator.py` (defaults: AAPL,MSFT,TSLA; 3 years history; 10,000 sims).
